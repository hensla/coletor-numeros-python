🔢 Coletor de Números Inteiros
Programa em Python que coleta números inteiros digitados pelo usuário e os armazena em uma lista, encerrando quando o valor 0 é inserido.

📋 Descrição
O programa solicita números inteiros um por vez, exibindo a posição de cada entrada (1°, 2°, 3°...). Conforme o usuário digita, os valores são armazenados em uma lista. Ao digitar 0, a coleta é encerrada e a lista completa é exibida.

🚀 Como usar
Pré-requisito: Python 3 instalado.
bashpython main.py
Exemplo de execução:
Informe o 1° número inteiro (para parar digite 0): 15
Informe o 2° número inteiro (para parar digite 0): 42
Informe o 3° número inteiro (para parar digite 0): 7
Informe o 4° número inteiro (para parar digite 0): 0
[15, 42, 7]

🛠️ Tecnologias

Python 3
